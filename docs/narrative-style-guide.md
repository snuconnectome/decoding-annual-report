# Narrative Style Guide — 과학적 발견 중심 · 차지욱 교수 말투 서술 원칙

> **2026-04-23 확립 (Phase 9)**. 평가단 대상 슬라이드·보고서 모두에 적용.

## 핵심 원칙 (두 축)

### 1. 과학적 발견 중심
본 과제는 **정량 벤치마크 성과 나열이 아니라 과학적 발견·기술 구축의 서사**로 제시한다. 평가단은 수치 자체가 아니라 **무엇을 알아냈고, 무엇을 만들었는지**를 판단한다.

### 2. 진중·섬세·조심스러운 말투 (차지욱 교수 스타일)
**AI 스타일의 성급한 일반화·자기과시는 금지**. 연구 결과를 겸손하고 객관적인 관찰자 관점으로 서술한다.

### 금지 표현 (AI hype 성급한 일반화)
- ❌ "세계 최초" · "국내 최초" · "최초" (독립 수식어로)
- ❌ "최고" · "압도" · "능가" · "뛰어넘음"
- ❌ "혁신적" · "획기적" · "돌파구"
- ❌ "완벽" · "완전" · "전면"
- ❌ "SOTA 달성" (헤드라인)
- ❌ "R² = 0.96 · 거의 전부 설명" 식 과장

### 문장 종결 규칙 (한국 R&D 연차보고서 관행)
**"...했다" 형이 아닌 "...함" 명사형 종결** 사용 (한국 정부 R&D 연차보고서 및 기술 문서 표준).

| 서술형 (지양) | 명사형 (권장) |
|---|---|
| 확립하였다 | 확립함 |
| 관찰되었다 | 관찰됨 |
| 보였다 | 보임 |
| 활용하였다 | 활용함 |
| 나타냈다 | 나타냄 |
| 시사한다 | 시사함 |
| 보고하였다 | 보고함 |
| 설계하였다 | 설계함 |
| 구축하였다 | 구축함 |
| 수행하였다 | 수행함 |
| 설명한다 | 설명함 |
| 검증한다 | 검증함 |
| 발견하였다 | 발견함 |
| 규명하였다 | 규명함 |

### 권장 표현 (진중한 관찰자 · 명사형 종결)
- ✅ "... 을 확립함", "... 을 보고함", "... 을 관찰함"
- ✅ "... 경향이 관찰되었으며, 이는 ... 을 시사함"
- ✅ "기존 연구는 ... 을 다루어 왔으나, 본 연구는 ... 을 ..."
- ✅ "... 수준의 성능을 보임" (구체 수치와 함께)
- ✅ "... 가능성을 시사함", "... 단서를 제공함"
- ✅ "... 선행 모델 대비 개선된 성능을 나타냄"
- ✅ "분야에서 보고된 바는 아직 제한적이나, 본 연구는 ..."

### 서술 톤 예시

**Before (AI hype)**: "세계 최초로 영화 시청 중 정서 변화를 뇌 활동만으로 초 단위 예측하는 모델 개발"

**After (진중 · 명사형)**: "영화 자극 시의 시간대별 정서가 변화를 개인 단위 fMRI 신호로부터 회귀하는 Sequence-to-Sequence 구조의 모델을 확립함"

**Before**: "두개내 뇌전도 분야 최초의 정량적 스케일링 법칙"

**After**: "두개내 뇌전도 데이터를 대상으로 한 사전학습의 정량적 스케일링 경향을 보고함. 언어모델에서 관찰되어 온 멱법칙 경향이 유사하게 나타났으며, 특히 제한된 연산 예산 하에서는 학습 반복 수가 모델 크기보다 성능 기여가 큰 것으로 관찰됨"

**Before**: "기존 LSTM이 사실상 예측 불가능하던 과제에서 정서 변동의 거의 전부를 설명"

**After**: "선형 반복 신경망 기반 베이스라인이 낮은 설명력 (R² < 0) 에 머물던 과제에서, 본 모델은 R² = 0.96 의 설명력을 보임"

## 3단 서술 구조 (Discovery → Construction → Evidence)

모든 연구 단위(실험·모델·프로토콜)는 다음 **세 가지 질문에 답하는 순서**로 기술:

1. **무엇을 알아냈는가 (Discovery)** — 뇌·행동·인지에 대한 **새로운 지식** 또는 인과 규명
2. **무엇을 만들었는가 (Construction)** — **신규 모델·프로토콜·데이터베이스** 의 정체성 (이름·구성·설계)
3. **그 증거는 무엇인가 (Evidence)** — 지표·수치·벤치마크는 여기에 배치 (주어가 아닌 **주어의 증거**)

## 적용 예시 (Before → After)

### Before (벤치마크 중심 — 지양)
> **SwiFT-IO**
> R² = 0.96
> Perceiver IO · Sequence-to-Sequence
> HBN movie fMRI · 677명 × 12분

### After (발견 중심 · 진중한 어조 · 명사형 종결 — 권장)
> **영화 자극 시의 연속 정서가 (emotion valence) 를 fMRI 신호로부터 개인 단위로 회귀하는 Sequence-to-Sequence 구조의 모델 (SwiFT-IO) 확립**
>
> 기존 파운데이션 모델 SwiFT 에 Perceiver IO 인코더를 결합하여 시계열 출력이 가능하도록 설계함. 선형 반복 신경망 기반 베이스라인이 R² 음수 수준에 머물던 과제에서, 본 모델은 R² = 0.96 의 설명력을 보임. 학습·검증에는 아동·청소년 코호트 Healthy Brain Network 의 영화 시청 fMRI (677명 × 각 약 12분) 를 활용함. 인지신경과학 학술대회 (CCN) 2025 발표.

### Before
> POYO-SSL: +12~13% 성능 향상 · SSIM 0.593

### After
> **동물 신경 세포의 반응 이질성을 대비 학습 신호로 활용하는 자기지도학습 프레임워크 (POYO-SSL) 구축 — 시각 자극을 보는 설치류의 영상 경험을 신경 활동으로부터 복원 가능함을 보고**
>
> 기존 자기지도학습은 신경 집단 내 무작위성을 잡음으로 처리하여 데이터 확장에 따른 성능 향상이 제한적이었음. 본 연구는 예측 가능한 세포 군집 (체세포 억제 뉴런 SST · 혈관 활성 장 펩타이드 뉴런 VIP · 파발부민 뉴런 PVALB · 뉴로텐신 수용체 뉴런 NTSR1) 을 우선 반영하도록 설계함. 초기 무학습 기반선 대비 12~13%p 의 성능 향상과 영화 프레임 복원 구조적 유사도 0.593 (1.0 이 완전 일치) 을 관찰함. Allen Brain Observatory 데이터로 검증 · ICLR 2026 심사 중.

### Before
> DIVER-1: 5.3k iEEG + 54k EEG hrs · 13M params → BrainBERT 43M 능가

### After
> **두개내·두피 뇌전도에 대한 대규모 사전학습의 정량적 스케일링 경향 보고 — 제한된 연산 예산에서 모델 크기보다 학습 반복을 우선하는 것이 효율적임을 관찰**
>
> 파운데이션 모델 DIVER-1 학습을 위해 두개내 뇌전도 5,300시간 및 두피 뇌전도 54,000시간 (총 17,700명 · 160만 채널-시간) 의 코퍼스를 구축함. 모델 크기·학습 반복·데이터 규모 간 관계에서 언어모델 분야에서 보고되어 온 멱법칙 경향이 유사하게 관찰됨. 1,300만 파라미터의 소형 모델이 선행 대형 모델 (BrainBERT 4,300만 · PopT 6,300만) 대비 동등하거나 개선된 성능을 보인 점은, 파라미터 수 확대보다 학습 체제 재설계의 중요성을 시사함. ICLR 2026 심사 중.

## 단어 선택 (Word Choice)

### 권장 (Discovery 언어)
- "… 을 규명함", "… 발견함", "… 입증함"
- "세계 최초로 …", "… 분야 최초로 …"
- "… 원리를 밝힘", "… 메커니즘을 드러냄"
- "… 가설을 인공지능으로 재현함"
- "… 연결성을 종 간에 잇는 …"

### 지양 (벤치마크 전면 언어)
- "X% 성능 향상" (헤드라인으로 단독 사용 금지)
- "SOTA 달성" (증거로만 사용)
- "R² = 0.96" (헤드라인 금지, 증거로 이관)
- "벤치마크 능가" (증거 맥락에서만)
