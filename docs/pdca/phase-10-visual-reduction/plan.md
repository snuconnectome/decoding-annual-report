## 축약 대상 (줄일 수 있는 것)

- 반복 수식어·연결어 ("선행 모델 대비", "일관되게", "체계적으로", "지속적으로")

## 슬라이드별 축약 전략

### S6 — 데이터베이스 구축 (최상 우선, 2,552 → 목표 1,300)
- **상단 animal block**: 좌우 2열 텍스트 → 중앙 큰 diagram (설치류 + 아레나 + 광섬유 + 도파민 센서) + 3개 핵심 bullet
- **하단 human block 좌 (실험 1)**: 7 bullet → 3 bullet + 뇌 영역 하이라이트 도식 (편도체·DMN·ACC 마킹된 뇌 이미지)
- **하단 human block 우 (실험 2)**: 8 bullet → 3 bullet + reactivation bias 개념 도식 (시간축 + 두 사건 재활성 그래프)
- 하단 캡션 유지

### S7A — 차지욱 Lab ① fMRI FM (상, 2,130 → 1,300)
- **2×2 카드**: 각 카드 6-7 bullet → 헤드라인 + 2 bullet + 카드 우상단 아이콘
- **카드 1 SwiFT**: "88억 파라미터" 배지 + 로그-로그 스케일링 소형 그래프 + 3단 확장 표기 (37M→8.8B)
- **카드 2 SwiFT-IO**: "R²=0.96" 게이지 + 영화 프레임 · 뇌 도식
- **카드 3 MBBN**: 주파수 대역 3층 도식 (고/저/초저) + ADHD/ASD 대비
- **카드 4 NeuroMamba**: 효율 개선 화살표 + HCP 94.9% 메트릭
- 하단 띠 TabLeT·LDT 간결화

### S7B — 차지욱 Lab ② EEG/iEEG + 임상 (최상 우선, 2,424 → 1,300)
- **Row 1 DIVER-1**: 기술 bullet 7개 → **3 bullet + DIVER 3-Phase 가로 다이어그램** (사전학습→정렬→개인화 블록)
- **Row 2 정렬**: Language-Brain + POYO-SSL subsection 각 3 bullet → 각 헤드라인 1줄 + 수치 배지 + 아이콘
- **Row 3 응용**: 3 칩 (RYM/CEBRA/Polygenic) 각 풍부 텍스트 → **각 칩에 아이콘 + 1줄**
- 하단 띠 협력자 명단 유지 (이미 축소됨)

### S8 — 문태섭 Lab SEED (상, 2,336 → 1,300)
- **상단 SEED 블록**: 메타 설명 4~5줄 → 2줄 (문제 + 해결책)
- **3 지표 카드**: 각 1-2 bullet 유지하되 아이콘 배지 (객체 · 텍스트 · 통합)
- **카드 하단 캡션** (선행 디코더 5개 나열): 아이콘 컬럼 + 연도만 (이름 유지)
- **Lower half**: VLM 패널 5 bullet → 3 bullet + 모델 도식. SEED 메타평가 패널 4 bullet → 2 bullet + 비교 막대 아이콘

### S9 — 종간 통합 (상, 2,270 → 1,300)
- **상단 공동연구 띠**: 2열 설명 → 중앙 다이어그램 (쥐 얼굴 + 편도체 뉴런 불꽃) + 1 문장
- **중단 좌 (동물) / 우 (사람)**: 각 7 bullet → 각 3 bullet + 데이터셋 배지 + 아이콘
- **하단 통합 띠**: 긴 서술 → 4 모달리티 아이콘 (sMRI·fMRI·iEEG/EEG·행동) 수렴 다이어그램 + 1 문장

### S10 — 응용·양자 (중, 1,420 → 1,000)
- **3 카드**: 각 3 bullet → 2 bullet + 큰 중앙 아이콘
- **양자 띠**: 1 칩 + 하단 캡션 → 양자 회로 소형 도식 + 1 문장

- 포커스: (1) 텍스트 축약 충분한가 (2) 아이콘·도식이 텍스트를 효과적으로 대체하는가 (3) 사실·수치 유지됐는가

**C. 15장 Gemini 재생성**

**D. PDF 재빌드 + 비교 시각 대조** (v0.9.1-verbose vs Phase 10)

**E. 커밋·푸시 + 태그** (v0.10.0-visual-reduced)

## Risks & Mitigation

| 리스크 | 대응 |
|---|---|
| Gemini 아이콘 렌더 품질 일관성 부족 | 각 프롬프트에 "flat line icon · no 3D · no photorealistic" 명시, 스타일 락 |
| 핵심 수치·발견 손실 | 보존 원칙 리스트 (위) 체크리스트로 관리 · 3회 리뷰 quality-engineer 담당 |
| Lab 기여 가시성 저하 | Lab 색 accent bar 유지 + 각 슬라이드 subtitle 에 Lab 명시 |
| 과도한 시각화 (평가단에 카툰 인상) | "진중한 학술 인포그래픽" 톤 유지 · 일러스트 스타일 아이콘 금지 |
